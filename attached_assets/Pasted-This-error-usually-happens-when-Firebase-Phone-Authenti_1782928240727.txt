This error usually happens when Firebase Phone Authentication tries to render the reCAPTCHA widget, but the DOM element it's supposed to attach to doesn't exist (or React has already unmounted it).

The key error is:

Cannot read properties of null (reading 'style')

This is almost always caused by one of these issues:

1. Missing reCAPTCHA container (most common)

Make sure your component contains a div for reCAPTCHA:

<div id="recaptcha-container"></div>

Then initialize it like this:

import { RecaptchaVerifier, getAuth } from "firebase/auth";

const auth = getAuth();

window.recaptchaVerifier = new RecaptchaVerifier(
  auth,
  "recaptcha-container",
  {
    size: "invisible",
  }
);

await window.recaptchaVerifier.render();
2. Component is unmounting

If you're using React Router and navigating away, the element is removed from the DOM while Firebase still tries to use it.

Create the verifier only once:

useEffect(() => {
  if (!window.recaptchaVerifier) {
    window.recaptchaVerifier = new RecaptchaVerifier(
      auth,
      "recaptcha-container",
      {
        size: "invisible",
      }
    );

    window.recaptchaVerifier.render();
  }
}, []);
3. React StrictMode

React 18's StrictMode mounts components twice in development, which can initialize reCAPTCHA twice.

Check if your main.tsx looks like:

<React.StrictMode>
  <App />
</React.StrictMode>

For testing, try removing StrictMode temporarily.

4. You're recreating the verifier on every click

❌ Don't do this:

const sendOTP = async () => {
    const verifier = new RecaptchaVerifier(...);
}

✅ Instead:

const verifier = window.recaptchaVerifier;

await signInWithPhoneNumber(auth, phone, verifier);
5. Vite hot reload

Your logs show:

[vite] hot updated

When Vite hot reloads, the old DOM is destroyed but Firebase still has a reference to the previous reCAPTCHA, causing this error.

A quick fix during development is to clear the old verifier before creating a new one:

if (window.recaptchaVerifier) {
    window.recaptchaVerifier.clear();
}

window.recaptchaVerifier = new RecaptchaVerifier(
    auth,
    "recaptcha-container",
    { size: "invisible" }
);

await window.recaptchaVerifier.render();
